# List Addons

List all available addons for the Teleshop store with their current status.

## Process

1. **List all addons:**
   ```
   list_workflows()
   ```

2. **Display as table:**
   Show ID, name, description, enabled/disabled status, last execution time.

## Example Usage
```
/list-addons
```
