# Get Memory

Retrieve a specific memory by its ID.

## Process

1. **Get memory:**
   ```
   get_memory({ memory_id: "<memory_id>" })
   ```

2. **Display full memory** details including content, metadata, and timestamps.

## Example Usage
```
/get-memory abc123-def456
```
